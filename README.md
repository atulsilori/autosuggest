# Auto Suggest App

This app suggest locations and hotel name based on your search string. In case of location it shows location name, map and address and in case of hotels it displays hotel name, description and address

## Run Locally

Clone the project

```bash
  git clone https://github.com/atulsilori/autosuggest.git
```

Go to the project directory

```bash
  cd autosuggest
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run ssr
```
